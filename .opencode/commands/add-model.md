description: إضافة مزود نموذج جديد (اسم + API key)
---

لإضافة مزود جديد:
1. اكتب `/connect` في الـ TUI
2. اختر المزود (Anthropic, OpenAI, Ollama, إلخ)
3. أدخل اسم المزود (أي اسم تختاره)
4. أدخل API Key الخاص بك
5. سيتم حفظه تلقائياً في auth.json

للتبديل بين النماذج بعد الإضافة:
- اكتب `/models` واختر النموذج المناسب
